# 到期提醒助手图标与启动页方向

> 创建日期：2026-05-05  
> 当前状态：视觉方向草稿，后续可交给 TRAE + Gemini 或图像生成工具出多版方案。

## 设计目标

1. 年轻、清爽、有秩序感。
2. 不像传统闹钟、日历、备忘录工具。
3. 能表达“快到期但不慌”的心理价值。
4. 在国内安卓应用市场的小图标尺寸下仍然清晰。

## 推荐方向

### 图标概念

用轻盈日历卡片叠加提醒光点，表达“快到期但不慌”。

画面可以由三层元素组成：

1. 圆角卡片：代表到期事项。
2. 小光点或提示泡：代表提醒。
3. 轻微折角或日期短线：代表时间感。

### 启动页概念

浅雾绿背景，中间是一张圆角到期卡片和一颗轻提示光点，文案保持克制。

推荐文案：

1. `到期提醒助手`
2. `把容易忘的小事，提前处理好`

## 推荐色彩

1. 清透青绿：`#1BAE9F`
2. 柔和橙：`#F5A24A`
3. 石墨灰：`#1F2A2A`
4. 浅雾绿：`#F7FAF9`

## 明确避免

1. 传统闹钟拟物图标
2. 大面积警告红
3. 老式日历翻页风
4. 廉价金币会员感
5. 过度商务化灰蓝风
6. 过度可爱化贴纸风
