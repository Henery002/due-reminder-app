## 基础信息

1. 应用名称：到期提醒助手
2. 副标题：会员续费、账单缴费、证件到期都不再临时想起
3. 短描述：订阅、账单、证件到期，能提醒，也能仅记录。
4. 推荐关键词：到期提醒、续费提醒、账单提醒、证件到期、会员续费
5. 包名：`com.henery.duereminderapp`

## 长描述草稿

到期提醒助手是一款专注管理生活到期事项的轻量工具。

你可以把视频会员、网盘会员、信用卡年费、房租、水电燃气、驾驶证、护照、资格证等容易忘的到期日放进一个清单里。应用会根据事项类型给出推荐提醒策略，并在到期前向系统安排本地通知；如果只是补录、留档或暂时不想收到通知，也可以保存为仅记录。

当前版本重点做好三件事：

1. 快速记录到期事项：选择类型和常用模板，确认到期日期、金额和备注。
2. 提醒方式更清楚：本地提醒用于提前处理，仅记录用于补录、留档或不想收到通知的事项。
3. 首页看清轻重缓急：今日、近 7 天、逾期事项优先展示。
4. 本地数据更安心：支持 JSON 文本备份和导入，暂不强制账号或云同步。

当前版本暂未接入广告、真实支付、账号登录或云同步。Pro 会员权益仅为后续预留，不展示虚假价格，也不提供假支付入口。

## 不建议使用的宣传点

1. 不写“永久不漏提醒”，因为 Android 通知会受系统权限、省电策略和后台限制影响。
2. 不写“云同步”，当前没有云服务。
3. 不写“会员已上线”或“订阅支付”，当前没有真实支付。
4. 不写“自动识别账单 / OCR”，当前没有 OCR 和 AI 提取。
5. 不写“官方证件提醒”，本应用只是个人辅助提醒工具。
6. 不写“已接入系统级保活”或“后台常驻”，当前只是向系统安排本地通知。
